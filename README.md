## Module Directory for Contao CMS

This module allows you to create compagnies regrouped by activities.

You can display the list of all compagnies and, by clicking on one of them, display its details.

The fields of a compagny are :
 - name
 - corporate_name
 - job
 - creation_date
 - address
 - zipcode
 - city
 - phone
 - mobile
 - email
 - activity
 - ape_code
 - description
 - website
 - logo
